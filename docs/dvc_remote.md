# DVC Remote Storage

The project uses a shared Google Drive folder as the default DVC remote.

## Purpose

The remote stores DVC-managed artifacts outside Git, including:

- Raw credit card transaction dataset
- Processed datasets generated by the pipeline
- Trained model output

## Configuration

The shared remote is configured in `.dvc/config`:

```bash
dvc remote add -d storage gdrive://1BO2yUzfJZrt7MXbkoxMWHeAwQmqOGA1e
